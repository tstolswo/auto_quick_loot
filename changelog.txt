# Auto Quick Loot - Changelog

## [1.3.1] - 2026-05-12
### Fixed
- Toggle key now correctly shows as "Toggle Quick Loot" in the Controls menu
  instead of the raw key ID `key.autoquickloot.toggle`.

### Added
- New "Edit HUD Position" keybinding (unbound by default). Press it in-game to open
  a drag-and-drop editor where you can reposition the HUD notice anywhere on screen.
  The position is saved to `autoquickloot-client.toml` when you close the editor.
- Added "Reset to Default" button in the HUD position editor.

## [1.3.0] - 2026-05-12
### Added
- Added **Toggle Mode**: a new `Auto Quick Loot: Toggle` keybinding (unbound by default, assign in Controls)
  that turns automatic looting on/off without needing to hold a key.
- Added a configurable HUD status indicator: when Toggle Mode is active, a small
  "Auto Quick Loot is enabled" notice appears in green text with a translucent background.
- Added client-side config file (`config/autoquickloot-client.toml`) with the following options:
  - `toggle.enabledByDefault` — start each session with Toggle Mode already on (default: false).
  - `hud.enabled`   — show/hide the HUD indicator (default: true).
  - `hud.anchor`    — screen corner for the indicator: TOP_LEFT, TOP_RIGHT, BOTTOM_LEFT, BOTTOM_RIGHT (default: BOTTOM_RIGHT).
  - `hud.offsetX`   — horizontal pixel offset inward from the chosen corner (default: 10).
  - `hud.offsetY`   — vertical pixel offset inward from the chosen corner (default: 10).

## [1.2.0] - 2026-05-06
### Added
- Added support for Loot Shulkers (`lootr:lootr_shulker`).

## [1.1.0] - 2026-05-06
### Added
- Added support for Lootr containers! The auto-loot keybinding will now automatically loot items from `lootr:lootr_barrel` in addition to `lootr:lootr_chest`.

### Changed
- Added `im_spot` to the mod authors list.
- General refactoring and update to the development environment.

## [1.0.0]
### Added
- Initial release.
- Added keybinding to automatically transfer items from vanilla chests and `lootr:lootr_chest` directly into the Mine and Slash backpack.
