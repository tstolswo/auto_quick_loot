# Auto Quick Loot

Auto Quick Loot is a Minecraft 1.20.1 Forge mod that automatically loots the contents of containers (like Chests, Lootr Chests, and Lootr Barrels) directly into your backpack when you open them. This mod is designed to work seamlessly with **Mine and Slash**.

## Features
- **Auto Looting**: Instantly transfers items from supported containers to your backpack.
- **Lootr Support**: Fully supports `lootr:lootr_chest` and `lootr:lootr_barrel`.
- **Keybinding**: Easily activate the auto-loot functionality with a dedicated keybind.

## Building from Source

This mod uses the standard Forge Gradle (MDK) environment.

### Prerequisites
- JDK 17
- Gradle

### Compile Instructions
To compile the mod yourself, open a terminal in the project root and run:
```bash
./gradlew build
```
The compiled `.jar` file will be located in `build/libs/`.
